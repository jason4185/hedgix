# Hedgix Market Protection Registry

The Hedgix Market Protection Registry defines the official protection products that can be purchased through the Hedgix GenLayer contract.

The registry is the source of truth for product terms:
- supported protected assets
- supported protection types
- event levels
- trigger rules
- premiums
- payouts
- durations
- Binance settlement symbols

Binance is the source of truth for market prices and historical settlement data. The registry defines what users are allowed to buy; Binance defines what happened in the market.
